# Mathematical Operations Quiz
## 1: You are a computer. What will this line of code print?
```
print(6 + 4 / 2 - (1 * 2))
```
**Result**
```
6.0
```

## 2: What is the data type of the result of the variable a in the following line of code:
```
a = int("5") / int(2.7)
```
**Result**
```
float
```
